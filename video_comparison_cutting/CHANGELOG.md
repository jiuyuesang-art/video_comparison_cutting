# 版本记录

## v0.5 - 当前版本

### 发布日期
2026-05-12

### 功能特性

#### 视频分析模式
- 支持三种分析模式：本地感知哈希、CLIP 视觉模型、云端模式
- 可调节灵敏度设置
- GPU 加速支持
- 支持多个原创视频同时比对

#### 导出功能
- 2560x1440 高清导出
- 左右分屏布局：
  - 左侧：剪辑视频（1538x865，位置 85,170）
  - 右侧：匹配片段（770x433，位置 1695,170）
- CRF 16 高质量编码
- 智能间隙填充（可配置阈值）
- 支持多种输出格式（MP4、MOV）
- 音频轨道保留

#### 用户界面
- 现代化 Web 界面
- 实时分析进度显示
- 视频预览功能
- 匹配结果可视化
- 导出进度追踪

#### 系统管理
- `setup.bat`：智能依赖安装
  - CLIP 模型缓存保护
  - 清洁安装模式 (`/clean`)
  - 跳过 FFmpeg 检查 (`/skip-ffmpeg`)
  - 依赖验证机制
- `start.bat`：启动管理
  - 端口清理
  - 上传存储自动清理
  - 浏览器自动打开

### 技术栈
- 前端：React + TypeScript + Zustand
- 后端：Node.js + Express + TypeScript
- 视频处理：FFmpeg + fluent-ffmpeg
- 图像处理：Sharp
- AI 模型：@xenova/transformers (CLIP)

### 缓存机制
| 类型 | 存储位置 | 清理方式 |
|------|----------|----------|
| 上传视频 | `/uploads/` | start.bat 启动时清理 |
| 导出视频 | `/exports/` | 手动清理 |
| CLIP 临时帧 | 系统临时目录 | 处理后立即删除 |
| 哈希临时帧 | 系统临时目录 | 处理后立即删除 |
